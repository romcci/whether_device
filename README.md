# Описание метеостанции 
## Устройство для определения локального тренда изменения погоды без выхода в интернет

Метеостанция показывает тренд изменения атмосферного давления за ближайший час,
а также выводит на экран текущую температуру, минимальную температуру за 
период от начала работы после сброса, текущую влажность. 
На основе этих показателей можно сделать предсказание об изменении 
погодных условий ближайшие несколько часов.

**Назначание:** карманный девайс для наблюдений за погодой в регионах где отсутствует интернет.
Его ценность в доступности, неприхотливости в работе, возможности самостоятельно проводить
наблюдения за погодными явлениями.

# Алгоритм работы:
1. выполняет измерение параметров текущией температуры, влажности, давления
2. записывает 20 измерений давления в буфер pressureHistory
3. каждый час выполняет расчет тренда давления на основе pressureHistory методом наименьших квадратов
4. выводит текстовый эквивалент текущего тренда исходя из 5 режимов: 
   1.5 → «Rast» (сильный рост).
   0.5 → «Rost» (рост).
   -0.5 → «Stab» (стабильно).
   -1.5 → «Pad» (падение).
   Иначе → «Spad» (сильное падение).

# Железо схемы:
- Arduino Nano type-C AtmegaOLD328
- GY-BME280 датчик температуры
- OLED 0.96 дисплей на I2C
- Кнопка тактовая DIP 12-12-7
- Аккумулятор 3,7 V Li-Pol 1000 mAh
- Модуль заряда с защитой TP4046 Type-C

# Внутри кода
- Версия с библиотекой Adafruit_SSD1306 для OLED

# Функционал:
- Выводит температуру, влажность, давление и высоту каждые 3 минуты
- Отслеживает минимальную температуру и сохраняет её в EEPROM
- Вычисляет барометрическую тенденцию за час
- Сброс минимальной температуры - удержание кнопки 3 секунды
- Включение/выключение экрана - короткое нажатие кнопки

# Распиновка:
**BME280:**
- VCC -> 3.3V (ВАЖНО! BME280 работает на 3.3V)
- GND -> GND
- SCL -> A5
- SDA -> A4

**OLED (I2C):**
- VCC -> 5V (или 3.3V)
- GND -> GND
- SCL -> A5 (общая шина с BME280)
- SDA -> A4 (общая шина с BME280)
 
**Кнопка:**
- Один контакт -> D2
- Другой контакт -> GND
